# Quiz-Game
Allows you to play multiple choice question answer quiz.

# Quiz game

# Requirements
1. Questions
2. options(4)
3. answer

# Functionalities:
4. timer
5. choose options
6. next
7. skip
8. score
9. exit
10. guest

# Phases:
1. display  questions & options(4) to the user
2. timer
3. take the answer from user
4. match the answers
5. score
6. ask user to go next or skip
